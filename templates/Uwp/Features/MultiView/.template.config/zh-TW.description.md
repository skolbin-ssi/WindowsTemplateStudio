讓您的使用者在個別視窗中檢視應用程式的獨立組件，幫助他們提高生產力。當您為應用程式建立多個視窗時，每個視窗都會有獨立的行為模式。工作列會個別顯示每個視窗。使用者可以獨立地移動、顯示、隱藏應用程式視窗及調整視窗大小，也可以在應用程式視窗之間切換，就像是個別應用程式一樣。每個視窗都會在其專屬的執行緒中運作。
